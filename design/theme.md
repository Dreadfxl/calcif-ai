## Typography
- Headers: Archivo Black, Roboto Condensed Bold, all-caps
- Body: IBM Plex Mono/Courier New, monospace
- Accent: Orbitron/Audiowide
- Letter-spacing: +2px to +5px (headers)
- Text-effects: Shadow, neon glow, chrome gradients

## UI Components
- Buttons: Chunky rectangles, hard edges, 3D box-shadow
- Cards: Strict rectangles, layered concrete texture
- Input Fields: Monospace, neon-highlight, large block cursor
- Sidebar/Navigation: Bold geometry, thick neon borders
- Dog Bone Icons: Low-poly, angular, chrome/neon states, glitch
